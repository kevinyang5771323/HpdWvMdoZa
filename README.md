# 学生宿舍管理系统 python1063

## 项目概述

学生宿舍管理系统是基于Python语言开发，结合了前端Vue框架和后端Django框架，使用Mysql数据库进行数据存储，旨在提高宿舍管理的效率与质量。

## 技术栈

- 后端：Python 3.7.7、Django
- 前端：Vue
- 数据库：Mysql 8
- 开发工具：PyCharm
- 数据库可视化工具：Navicat

## 系统架构

- 架构模式：MVC（Model-View-Controller）

## 运行环境

- 操作系统：Windows 10、MacOs
